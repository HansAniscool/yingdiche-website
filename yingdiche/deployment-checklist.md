## 🔄 待完成的任务

### 第一步：选择部署方案
- [ ] **方案A**：对象存储（推荐新手）
  - 阿里云OSS 或 腾讯云COS
  - 配置简单，无需服务器管理
  - 支持CDN加速

- [ ] **方案B**：虚拟主机
  - 购买虚拟主机服务
  - 通过FTP上传文件

- [ ] **方案C**：云服务器（VPS）
  - 完全控制权
  - 需要配置Nginx

### 第二步：域名解析配置
- [ ] 登录域名注册商后台
- [ ] 添加A记录：`@` → `服务器IP`
- [ ] 添加A记录：`www` → `服务器IP`
- [ ] 等待DNS生效（通常10-60分钟）

### 第三步：文件上传
- [ ] 将整个 `yingdiche` 文件夹上传到服务器
- [ ] 确保文件权限正确（755 for 目录，644 for 文件）
- [ ] 测试首页访问

### 第四步：SSL证书配置
- [ ] 申请免费SSL证书（Let's Encrypt）
- [ ] 配置HTTPS重定向
- [ ] 测试HTTPS访问

### 第五步：最终测试
- [ ] 测试所有页面链接
